# Alowishus Delicious

Alowishus Delicious is a simple, responsive and accessible coffee shop landing page built using HTML, CSS & JavaScript.

## Features

- Responsive design (320px+)
- Accessible to specially abled people
- Smooth scroll animations

## Tech Stack

- HTML
- CSS
- JavaScript
- Swiper.js
